"Active contours"  Help page.

This GUI goal is to supply the user intuitive and easy to use tool to get acquainted and to compare several snakes methods for object segmentation/boundaries detection.

For basic GUI use of this tool please follow those instructions:

  1.  Choose image file for segmentation using Load Image File button (some test images are supplied in the Images library).

  2.  Choose noise model parameters in  Noise Model panel.

  3. Choose the Gaussian blur filer parameters (N,sigma) in  Blur (denoise) panel (default values are N=16, sigma=0).

  4. Choose the Color space and color dimension in  Color space panel (default choice is YCbCr and Y).

  5. Choose the contour based segmentation algorithm to apply-using the Applied Algorithm panel.

  6. Set desired algorithm parameters- using the Alg. parameters panel.

  7. Choose your display settings using the Display Menu panel.

  8. Choose refresh rate of displayed images and save rate of saved images  - using the appropriate edit button in the I/O panel.

  9. Choose output path for saved images during algorithm application (default - current directory).

  10. Choose number of iterations suiting your needs and Finally press Run to enjoy the magic .

Advanced use will involve:

  1.  Modification of above algorithms.

  2. Adding new algorithms to the GUI.

  3. Comparing algorithms, especially in cases where some will fail.

  4. And more...
